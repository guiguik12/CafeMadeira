# Aura Cup — Menu Digital

Aura Cup — Menu Digital é uma aplicação web moderna e baseada em seções que apresenta a experiência de uma cafeteria através de uma interface digital interativa.

A identidade visual é inspirada no premiado projeto de branding "Aura Cup I Coffee" disponível em [Behance](https://www.behance.net/gallery/245065107/AURA-CUP-I-COFFEE-I-brand-identity).

## Objetivos

- Criar uma experiência multi-seção envolvente que showcaseia a marca da cafeteria  
- Implementar animações sofisticadas e efeitos visuais (shaders Plasma, efeitos de blur em texto, efeitos brilhantes)  
- Entregar um design totalmente responsivo com abordagem mobile-first  
- Demonstrar arquitetura moderna de frontend com design baseado em componentes e gerenciamento de estado  
- Traduzir branding premium em um produto digital interativo  

## Inspiração & Branding

O design é inspirado em *"AURA CUP"* por Богдана Собко [Behance](https://www.behance.net/gallery/245065107/AURA-CUP-I-COFFEE-I-brand-identity).

O conceito enfatiza o café como um momento de pausa e reflexão, expresso através de minimalismo, elementos visuais suaves e uma paleta de cores refinada.

Todas as decisões de UI (tipografia, cores, espaçamento) foram adaptadas deste conceito em uma interface digital funcional.

## Motivação

Este projeto explora a transformação de branding premium em uma experiência web interativa.  
Serve como um portfólio que demonstra expertise em:

- Padrões avançados de React (Context API, Hooks)  
- Animações complexas (timelines GSAP, Framer Motion)  
- Programação de gráficos WebGL (efeitos de shader OGL)  
- Design responsivo e mobile-first  
- Ferramentas de build modernas e otimização  
- Arquitetura de componentes e organização de código  

## Funcionalidades Principais

- **Navegação Multi-Seção**: Transições suaves entre as seções Menu, About, Galeria, Depoimentos e Localização
- **Animações Avançadas**: 
  - Efeitos de shader Plasma (com poder de WebGL)
  - Animações de blur reveal em texto (componente BlurText)
  - Efeitos de texto brilhante (componente ShinyText)
  - Menu de navegação baseado em timeline GSAP
  - Animações de hover com física de spring
- **Design Responsivo**: Abordagem mobile-first com breakpoints otimizados
- **Gerenciamento de Estado**: React Context API para roteamento baseado em seções
- **Componentes Interativos**: Galeria circular, cards de menu com efeitos de hover, exibição dinâmica de preços
